📚 算法编写指南

在编写寻路算法时，您可以使用以下预定义的变量和函数：

可用变量：

  • grid - 二维数组，表示网格状态 (0: 空地, 1: 障碍物, 2: 起点, 3: 终点)
  • start_pos - 起点坐标 [y, x]
  • end_pos - 终点坐标 [y, x]
  • width - 网格宽度
  • height - 网格高度
  • diagonal - 是否允许对角线移动 (布尔值)

辅助函数：

  • get_neighbors(y, x) - 获取指定位置的邻居节点坐标列表
  • heuristic(y1, x1, y2, x2) - 计算两点间的启发式距离
  • is_valid_position(y, x) - 检查位置是否有效

返回格式：

您的算法需要返回一个包含以下信息的字典：

  • visited - 访问过的节点坐标列表 [[y, x], [y, x], ...]
  • path - 找到的路径坐标列表 [[y, x], [y, x], ...]
  • found - 是否找到路径 (布尔值)

示例算法结构：

class CustomPathfindingAlgorithm:
    def __init__(self, width: int, height: int):
        self.width = width
        self.height = height
        self.visited_order = []

    def find_path(self, grid, start, end) -> List[Tuple[int, int]]:
        """
        主要寻路方法

        参数:
            grid: 二维数组，0=空地, 1=障碍物, 2=起点, 3=终点
            start: 起点坐标 (y, x)
            end: 终点坐标 (y, x)

        返回:
            路径坐标列表 [(y1, x1), (y2, x2), ...]
        """
        self.visited_order = []  # 重置访问顺序

        # 实现您的寻路算法...
        # 例如：BFS, DFS, A*, Dijkstra等

        return []  # 返回找到的路径

    def get_visited_order(self) -> List[Tuple[int, int]]:
        """
        获取访问节点的顺序（用于可视化）

        返回:
            访问顺序列表 [(y1, x1), (y2, x2), ...]
        """
        return self.visited_order

🔧 AI生成算法故障排除

如果AI生成的算法无法运行，通常是因为以下原因：

常见问题及解决方案：

  • 网格数据类型错误：AI可能假设grid包含CellType枚举，但实际传入的是整数 
    # 错误写法：
if grid[y][x] != CellType.WALL:  # CellType不存在

# 正确写法：
if grid[y][x] != 1:  # 1 表示障碍物
  • 坐标系统混淆：系统使用 (y, x) 格式，AI可能使用 (x, y) 
    # 正确的坐标格式：
start = (y, x)  # 行, 列
end = (y, x)    # 行, 列
  • visited_order使用错误 # 在find_path开始时重置： self.visited_order = [] # 访问节点时添加： self.visited_order.append((y, x))
  • 方法签名不匹配：确保类和方法签名正确 
    class CustomPathfindingAlgorithm:
    def __init__(self, width: int, height: int):
        self.width = width
        self.height = height
        self.visited_order = []

    def find_path(self, grid, start, end) -> List[Tuple[int, int]]:
        # 实现算法...

    def get_visited_order(self) -> List[Tuple[int, int]]:
        return self.visited_order
  • 复杂的算法实现：避免过于复杂的高级算法，从简单BFS开始 
    # 推荐使用简单可靠的算法：
- BFS（广度优先搜索）
- DFS（深度优先搜索）
- 简化的A*算法

# 避免过于复杂的算法：
- Jump Point Search（容易出错）
- 复杂的混合算法
- 需要大量数学计算的算法

建议解决方案：

  1. 使用页面提供的标准BFS模板作为基础
  2. 查看示例算法中的A*、Dijkstra、BFS实现
  3. 先用简单的测试用例验证算法
  4. 逐步添加复杂功能，确保每步都正确
  5. 如果AI生成的代码有问题，可以参考示例算法手动修正

📖 示例算法

A* 算法

A*算法是一种启发式搜索算法，它结合了Dijkstra算法的准确性和贪心算法的效率。

查看完整代码 
def find_path(grid, start_pos, end_pos, width, height, diagonal=False):
    import heapq
    import math

    def heuristic(y1, x1, y2, x2):
        if diagonal:
            return math.sqrt((y1-y2)**2 + (x1-x2)**2)
        else:
            return abs(y1-y2) + abs(x1-x2)

    def get_neighbors(y, x):
        neighbors = []
        directions = [(-1,0), (1,0), (0,-1), (0,1)]
        if diagonal:
            directions.extend([(-1,-1), (-1,1), (1,-1), (1,1)])

        for dy, dx in directions:
            ny, nx = y + dy, x + dx
            if 0 <= ny < height and 0 <= nx < width and grid[ny][nx] != 1:
                neighbors.append((ny, nx))
        return neighbors

    start_y, start_x = start_pos
    end_y, end_x = end_pos

    # 优先队列：(f_score, g_score, y, x, parent)
    heap = [(heuristic(start_y, start_x, end_y, end_x), 0, start_y, start_x, None)]

    # 记录访问信息
    visited = []
    visited_set = set()

    # 记录每个位置的父节点和g值
    parent = {}
    g_score = {start_pos: 0}
    f_score = {start_pos: heuristic(start_y, start_x, end_y, end_x)}

    while heap:
        current_f, current_g, y, x, current_parent = heapq.heappop(heap)
        current_pos = (y, x)

        if current_pos in visited_set:
            continue

        visited_set.add(current_pos)
        visited.append([y, x])

        if current_pos == end_pos:
            # 找到终点，重建路径
            path = []
            while current_pos in parent:
                path.append(list(current_pos))
                current_pos = parent[current_pos]
            path.append(list(start_pos))
            path.reverse()

            return {
                'visited': visited,
                'path': path,
                'found': True
            }

        for ny, nx in get_neighbors(y, x):
            neighbor_pos = (ny, nx)
            tentative_g = current_g + (1.414 if diagonal else 1)

            if neighbor_pos not in g_score or tentative_g < g_score[neighbor_pos]:
                parent[neighbor_pos] = current_pos
                g_score[neighbor_pos] = tentative_g
                f_score[neighbor_pos] = tentative_g + heuristic(ny, nx, end_y, end_x)
                heapq.heappush(heap, (f_score[neighbor_pos], tentative_g, ny, nx, current_pos))

    return {
        'visited': visited,
        'path': [],
        'found': False
    }

Dijkstra 算法

Dijkstra算法是一种经典的单源最短路径算法，保证找到最优路径。

查看完整代码 
def find_path(grid, start_pos, end_pos, width, height, diagonal=False):
    import heapq

    def get_neighbors(y, x):
        neighbors = []
        directions = [(-1,0), (1,0), (0,-1), (0,1)]
        if diagonal:
            directions.extend([(-1,-1), (-1,1), (1,-1), (1,1)])

        for dy, dx in directions:
            ny, nx = y + dy, x + dx
            if 0 <= ny < height and 0 <= nx < width and grid[ny][nx] != 1:
                neighbors.append((ny, nx))
        return neighbors

    start_y, start_x = start_pos
    end_y, end_x = end_pos

    # 优先队列：(distance, y, x, parent)
    heap = [(0, start_y, start_x, None)]

    # 记录访问信息
    visited = []
    visited_set = set()

    # 记录最短距离和父节点
    distance = {start_pos: 0}
    parent = {}

    while heap:
        current_dist, y, x, current_parent = heapq.heappop(heap)
        current_pos = (y, x)

        if current_pos in visited_set:
            continue

        visited_set.add(current_pos)
        visited.append([y, x])

        if current_pos == end_pos:
            # 找到终点，重建路径
            path = []
            while current_pos in parent:
                path.append(list(current_pos))
                current_pos = parent[current_pos]
            path.append(list(start_pos))
            path.reverse()

            return {
                'visited': visited,
                'path': path,
                'found': True
            }

        for ny, nx in get_neighbors(y, x):
            neighbor_pos = (ny, nx)
            edge_weight = 1.414 if diagonal else 1
            tentative_dist = current_dist + edge_weight

            if neighbor_pos not in distance or tentative_dist < distance[neighbor_pos]:
                distance[neighbor_pos] = tentative_dist
                parent[neighbor_pos] = current_pos
                heapq.heappush(heap, (tentative_dist, ny, nx, current_pos))

    return {
        'visited': visited,
        'path': [],
        'found': False
    }

广度优先搜索 (BFS)

广度优先搜索是一种简单的图遍历算法，一层一层地搜索，保证找到最短路径（在无权图中）。

查看完整代码 
def find_path(grid, start_pos, end_pos, width, height, diagonal=False):
    from collections import deque

    def get_neighbors(y, x):
        neighbors = []
        directions = [(-1,0), (1,0), (0,-1), (0,1)]
        if diagonal:
            directions.extend([(-1,-1), (-1,1), (1,-1), (1,1)])

        for dy, dx in directions:
            ny, nx = y + dy, x + dx
            if 0 <= ny < height and 0 <= nx < width and grid[ny][nx] != 1:
                neighbors.append((ny, nx))
        return neighbors

    start_y, start_x = start_pos
    end_y, end_x = end_pos

    # 队列：(y, x, parent)
    queue = deque([(start_y, start_x, None)])

    # 记录访问信息
    visited = []
    visited_set = set([(start_y, start_x)])

    # 记录父节点
    parent = {}

    while queue:
        y, x, current_parent = queue.popleft()
        current_pos = (y, x)

        visited.append([y, x])

        if current_pos == end_pos:
            # 找到终点，重建路径
            path = []
            while current_pos in parent:
                path.append(list(current_pos))
                current_pos = parent[current_pos]
            path.append(list(start_pos))
            path.reverse()

            return {
                'visited': visited,
                'path': path,
                'found': True
            }

        for ny, nx in get_neighbors(y, x):
            neighbor_pos = (ny, nx)
            if neighbor_pos not in visited_set:
                visited_set.add(neighbor_pos)
                parent[neighbor_pos] = current_pos
                queue.append((ny, nx, current_pos))

    return {
        'visited': visited,
        'path': [],
        'found': False
    }
